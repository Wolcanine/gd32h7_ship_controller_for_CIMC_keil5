  【9.4 PS2 遥控映射】
    ↑         油门 +1%   (throttle +0.01, max 0.10)
    ↓         油门 -1%   (throttle -0.01, min 0.00)
    ←         差速左偏   (diff -0.01, min -0.10)
    →         差速右偏   (diff +0.01, max +0.10)
    △         紧急停车   (throttle=0, diff=0)
    ×         差速归零   (diff=0)
    □         油门 +5%   (快速步进)
    ○         油门 -5%   (快速步进)

    注：按键上升沿检测，按住不放不会重复触发（50Hz 主循环）。

  【9.5 电机 duty 计算】
    left_duty  = clamp(throttle + diff,  0.0, 0.10)
    right_duty = clamp(throttle - diff,  0.0, 0.10)
    仅正向驱动 [0.00, 0.10]，不反转。

  【9.6 安全设计】
    - 上电刹车：pwm_output_init() 默认 IN1=0 IN2=0 (刹车)
    - 仅正向驱动：duty 钳位 [0.00, 0.10]，不会反转
    - 限幅 10%：远低于 PWM_MAX_DUTY (25%)
    - △ 一键急停：throttle 和 diff 同时归零
    - 方向突变刹车：pwm_output.c 内置 100ms 刹车保护
    - 按键上升沿触发：防止持续按住导致意外加速

  【9.7 串口输出】
    初始化阶段：打印模块状态 + 按键映射表 + 引脚位置
    运行阶段：仅在 duty 变化时输出一行
      格式: MOTOR: T=+0.05 D=+0.02 | L=+0.07 R=+0.03
    LED1 (PF10) 以 2.5Hz 闪烁作为心跳

  四、无负载 PWM 测试（驱动板不上电，仅测信号）
    1. 万用表电压档量各 IN1/IN2 引脚 → 应全为 0V（刹车状态）
    2. 按 ↑ 一下 → 串口应输出 "MOTOR: T=+0.01 D=+0.00 | L=+0.01 R=+0.01"
    3. 万用表量左 IN1 (PC3) → 应为 3.3V
       左 IN2 (PC5) → 应为 0V
       右 IN1 (PF9) → 应为 3.3V
       右 IN2 (PD2) → 应为 0V
    4. 示波器量 ENA 引脚 PWM → 占空比约 4% (0.01/0.25)
    5. 按 ↑ 多次逐步增加油门，PWM 占空比应随之增大
    6. 按 × 差速归零，按 ←/→ 测试差速
    7. 按 △ 急停 → 所有输出归零

  五、带载测试（驱动板上电，电机接上）
    1. 确认电机空载、无人靠近、可随时断电
    2. 驱动板上电
    3. 按 ↑ 一下 → 电机应以极低速度转动（油门 1%）
    4. 逐步增加油门到 5%，确认转速随之增加
    5. 测试差速 ←/→ 确认左右电机速度差异
    6. 测试 △ 急停 → 电机立即停止
    7. 确认正转方向：船模前进 = 左右电机同向正转
       如方向反了，对调对应电机的两根线即可

  六、异常处理
    - 上电电机就转 → 检查共地、检查 IN1/IN2 电平
    - 电机不转 → 检查驱动板电源、ENA PWM 信号、IN1/IN2 电平
    - 占空比不对 → 示波器量 ENA，确认 pwm_output 正常
    - PS2 无响应 → 检查手柄连接、电池
